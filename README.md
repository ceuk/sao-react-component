# sao-react-component

> Simple React component with scss module

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao react-component my-project
```

### From git

```bash
sao ceuk/sao-react-component my-project
```
